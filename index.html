KPI Definitions

SAMPLE - REDACTED FOR PORTFOLIO

Standardized KPI Rules (Sample Extract)

Clear definitions prevent "where did this number come from?" moments.

  • Qualified Lead: Contact with verified email + interest signal (form fill, demo request)
  • Conversion: Completed target action (varies by campaign goal)
  • Cost per Acquisition: Total spend ÷ qualified conversions (not clicks)
  • Attribution Window: 7-day click, 1-day view (standardized across reports)
  • Active Campaign: Live with spend in reporting period

UTM Template

SAMPLE - REDACTED FOR PORTFOLIO

UTM Naming Standard (Sample)

Consistent structure enables clean campaign comparison and reporting.

  • utm_source: Platform (google / facebook / linkedin / email)
  • utm_medium: Traffic type (cpc / social / email / organic)
  • utm_campaign: Campaign identifier (q1-2024-brand-awareness)
  • utm_content: Creative variant (banner-a / video-intro / headline-test)
  • utm_term: Keyword (paid search only)

Example URL

example.com/landing?utm_source=google&utm_medium=cpc&utm_campaign=q1-2024-product-launch&utm_content=headline-test-a
